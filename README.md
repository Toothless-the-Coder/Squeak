# Squeak – Custom Bambu Labs Mouse

##  Project Description

Squeak is a custom-designed computer mouse built using components from a Bambu Labs kit. This project combines 3D printed parts, embedded electronics, and firmware to create a fully functional, compact pointing device.

The mouse features a custom ergonomic shell, integrated electronics, and a modular internal layout designed for easy assembly and iteration.

---

## How to Use

1. 3D print all CAD components (see `/CAD` folder)
2. Assemble the nouse
3. Connect the mouse to your computer via USB dongle
4. Use like a standard mouse!

---

## Why I Made This Project

I created Squeak to explore custom hardware design using Bambu Labs components. The goal was to design a fully functional peripheral from scratch while learning more about CAD, electronics integration, and firmware development.

This project is also meant to demonstrate how consumer fabrication tools can be used to create practical, everyday devices.

---

## Design Overview

* Fully custom CAD assembly including all components
* Designed for manufacturability using 3D printing
* Integrated electronics from Bambu Labs kit
* Compact internal layout for efficient space usage
